# Simulador-de-Investimentos-de-fundos-imobilliarios
Simulador de investimentos em Fundos de Investimento Imobiliário (FIIs), desenvolvido com finalidade educacional, destinado a demonstrar conceitos básicos de alocação de capital, aportes periódicos e estimativa de renda passiva para investidores iniciantes. Este simulador não constitui recomendação de investimento de qualquer natureza.
